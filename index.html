O Projeto

A Thigas Automation Systems está desenvolvendo uma infraestrutura escalável para processamento de documentos e automação de fluxos de RH e Logística.

Nossa Stack Tecnológica:

  • Linguagem: Python 3.x
  • Integrações: Google Apps Script, Trello API, PDF Processing Libraries
  • Objetivo AWS: Migrar processamento local (On-premise) para AWS Lambda e EC2 para ganho de performance e redução de latência.

Por que AWS?

Buscamos os créditos do AWS Activate Founders para escalar nossa capacidade de processamento de PDFs em lote e garantir alta disponibilidade nas automações de workflow que hoje rodam em ambientes limitados.
